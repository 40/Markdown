jekyll を使ったページのレイアウト
============================

jekyll とは？
------------
テンプレートから HTML を生成する Ruby プログラム

### インストール ###
[RubyGems を用いてインストール](https://github.com/mojombo/jekyll/wiki/Install)

	gem install jekyll

### 構成ファイル ###
[基本構成](https://github.com/mojombo/jekyll/wiki/Usage) は以下の通り。

	.
	├─ _config.yml
	├─ _includes
	├─ _layouts
	│   └─ default.html
	├─ _site
	└─ index.md

+ **_config.yml**  

+ **_includes**  

+ **_layouts**  

+ **_site**  

+ **index.md**
